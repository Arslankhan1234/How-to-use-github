Following steps should follow for using git and github

  1. Create a folder in your directory
  2. Right click inside folder and open git bash or cmd
  3. Type git init to convert directory into git repository
  4. Create your files and add in your folder
  5. Type git status in your cmd to check files status
  6. Type git add filename to add untrack files into staging area
  7. Git commit -m message to commit the files

After committing all files steps to upload on github

  1. Create Account on git hub if not already exist by github sign up
  2. After creating account sign in to account using email and password
  3. After signin click on new button at top left side of github
